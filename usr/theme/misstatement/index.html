API 文档
  • GET /api
    获取服务状态信息
  • GET /api/img/types
    获取所有可用的图片类型
  • GET /random_image/<type>
    直接返回随机图片 (二进制流)
  • GET /random_image/j/<type>
    返回图片信息的JSON
  • GET /random_image/g/<type>
    重定向到图片URL (302)
  • GET /api/img/<type>/count
    获取指定类型的图片数量统计
